### [Monotonic Array](https://leetcode.com/problems/monotonic-array)

<p>An array is <em>monotonic</em> if it is either monotone increasing or monotone decreasing.</p>

<p>An array <code>nums</code> is monotone increasing if for all <code>i &lt;= j</code>, <code>nums[i] &lt;= nums[j]</code>.&nbsp; An array <code>nums</code> is monotone decreasing if for all <code>i &lt;= j</code>, <code>nums[i] &gt;= nums[j]</code>.</p>

<p>Return <code>true</code> if and only if the given array <code>nums</code> is monotonic.</p>
