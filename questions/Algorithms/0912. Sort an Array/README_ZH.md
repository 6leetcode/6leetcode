### [排序数组](https://leetcode-cn.com/problems/sort-an-array)

<p>给你一个整数数组&nbsp;<code>nums</code>，请你将该数组升序排列。</p>

<p>&nbsp;</p>

<ol>
</ol>

<p><strong>示例 1：</strong></p>

<pre><strong>输入：</strong>nums = [5,2,3,1]

</pre>

<pre><strong>输入：</strong>nums = [5,1,1,2,0,0]
