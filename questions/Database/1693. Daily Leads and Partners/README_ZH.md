### [每天的领导和合伙人](https://leetcode-cn.com/problems/daily-leads-and-partners)

<p>表：<code>DailySales</code></p>

<pre>+-------------+---------+
| Column Name | Type    |
+-------------+---------+
| date_id     | date    |
| make_name   | varchar |
| lead_id     | int     |
| partner_id  | int     |
+-------------+---------+
该表没有主键。
该表包含日期、产品的名称，以及售给的领导和合伙人的编号。
名称只包含小写英文字母。</pre>

<p> </p>

<p>写一条 SQL 语句，使得对于每一个 <code>date_id</code> 和 <code>make_name</code>，返回<strong>不同</strong>的 <code>lead_id</code> 以及<strong>不同</strong>的 <code>partner_id</code> 的数量。</p>

<p>按任意顺序返回结果表。</p>

<p>查询结果格式如下示例所示：</p>

<p> </p>
