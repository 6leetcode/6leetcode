### [Base 7](https://leetcode.com/problems/base-7)

<p>Given an integer, return its base 7 string representation.</p>

<p><b>Example 1:</b><br />
<pre>
<b>Input:</b> 100
<b>Output:</b> "202"
</pre>
</p>

<p><b>Example 2:</b><br />
<pre>
<b>Input:</b> -7
<b>Output:</b> "-10"
</pre>
</p>

<p><b>Note:</b>
The input will be in range of [-1e7, 1e7].
</p>
