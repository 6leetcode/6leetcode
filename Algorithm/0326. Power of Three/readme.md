### [Power of Three](https://leetcode.com/problems/power-of-three/)

Given an integer, write a function to determine if it is a power of three.

__Example 1__:

```
Input: 27
Output: true
```

__Example 2__:

```
Input: 0
Output: false

```

__Example 3__:

```
Input: 9
Output: true
```
__Example 4__:

```
Input: 45
Output: false
```

__Follow up__:

Could you do it without using any loop / recursion?
