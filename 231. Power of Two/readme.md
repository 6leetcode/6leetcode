### [Power of Two](https://leetcode.com/problems/power-of-two/)

Given an integer, write a function to determine if it is a power of two.

__Example 1:__

```
Input: 1
Output: true 
Explanation: 2**0 = 1
```

__Example 2:__

```
Input: 16
Output: true
Explanation: 2**4 = 16
```

__Example 3:__

```
Input: 218
Output: false
```
